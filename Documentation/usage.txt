Documentation for usage of code for Travelling Salesman Problem using Genetic Algorithm

Usage :
Enter input in the following format in the "input.txt" file in directory
Integer N -> Number of cities
Integer E -> Number of edges
Edges in format -> Start City End City Distance

Note : Cities are numbered from 0 to N-1

Constraints : 
	1. Graph should be an undirected graph with no negative edges
	2. Number of edges E should not exceed the Maximum possible number of edges for an undirected graph i.e E <= N*(N-1)/2

Example: 
4 4
0 1 5
1 3 6
0 3 9
0 2 4
